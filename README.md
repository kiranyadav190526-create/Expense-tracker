# 💰 Expense Tracker Pro

A modern and responsive Expense Tracker built with HTML, CSS, and JavaScript to help users manage their personal finances efficiently.

## 🚀 Features

* Add, Edit, and Delete Transactions
* Income & Expense Tracking
* Real-Time Balance Calculation
* Monthly Expense Summary
* Search Transactions
* Filter by Category
* Expense Pie Chart using Chart.js
* Export Transactions to CSV
* Dark Mode Support
* Data Persistence with Local Storage
* Responsive Design for Mobile & Desktop

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Chart.js
* Local Storage API

## ⚡ Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/expense-tracker-pro.git
```

2. Open the project folder

```bash
cd expense-tracker-pro
```

3. Run `index.html` in your browser.

## 📊 Functionality

Users can manage income and expenses, visualize spending patterns through charts, search and filter transactions, and export data for future use. All records are automatically saved in the browser using Local Storage.
